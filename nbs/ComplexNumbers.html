Complex Numbers¶

A complex number is an ordered pair \((a, b)\), where \(a, b \in \mathbb{R}\). This is written \(a + bi\) where \(i \equiv \sqrt{-1}\) (Leonhard Euler, 1777) s.t. \(0 + 1i = i\).
The set of all complex numbers is denoted \(\mathbb{C} = \{a + bi : a, b \in \mathbb{R}\}\).

\(i \equiv \sqrt{-1}\)
\(i^2 = \sqrt{-1}\sqrt{-1} = -1\)
\(i^3 = ii^2 = -i\)
\(i^4 = ii^3 = -ii = -i^2 = 1\)

Real Numbers¶

\(\mathbb{R} \subset \mathbb{C}\)
\(a + 0i = a\) where \(a \in \mathbb{R}\)

Imaginary Numbers¶

\(\mathbb{I} \subset \mathbb{C}\)
\(0 + bi = bi\) where \(b \in \mathbb{R}\)

Commutativity¶

\(\alpha + \beta = \beta + \alpha\)
\(\alpha\beta = \beta\alpha\)
\(\forall \alpha, \beta \in \mathbb{C}\)

Associativity¶

\((\alpha + \beta) + \lambda = \alpha + (\beta + \lambda)\)
\((\alpha\beta)\lambda = \alpha(\beta\lambda)\)
\(\forall \alpha, \beta, \lambda \in \mathbb{C}\)

Existence of Identity Element¶

\(\lambda + 0 = \lambda\)
\(\lambda 1 = \lambda\)
\(\forall \lambda \in \mathbb{C}\)

Existence of Inverse Elements¶

\(\forall \alpha \in \mathbb{C}\), \(\exists \beta \in \mathbb{C}\) s.t. \(\alpha + \beta = 0\)
The additive inverse of an element \(\alpha\) is denoted \(-\alpha\).
\(\forall \alpha \in \mathbb{C}\) with \(\alpha \neq 0\), \(\exists \beta \in \mathbb{C}\) s.t. \(\alpha\beta = 1\)
The multiplicative inverse of an element \(\alpha \neq 0\) is denoted \(\alpha^{-1}\).

Distributivity¶

\(\lambda(\alpha + \beta) = \lambda\alpha + \lambda\beta\)
\(\forall \lambda, \alpha, \beta \in \mathbb{C}\)

Complex Substraction¶

\(\beta - \alpha \equiv \beta + (-\alpha)\)

Complex Division¶

\(\frac{\beta}{\alpha} \equiv \beta\alpha^{-1}\)
